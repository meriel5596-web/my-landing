# MERIELINVENTORY - 김희정 물류 전문가 프로필 사이트

**글로벌 물류의 전문가, 김희정과 함께하세요**

20년 이상의 경력을 가진 물류 전문가 김희정의 프로필 및 서비스 홍보 사이트입니다. 해상 수출입, 컨테이너 관리, FTA 및 3PL 전문 컨설팅과 교육을 제공합니다.

---

## 🌟 프로젝트 개요

MERIELINVENTORY는 물류 전문가 김희정 강사의 경력, 전문 분야, 프로젝트 레퍼런스, 고객 리뷰 및 문의를 관리하는 전문적인 홍보 웹사이트입니다.

### 주요 특징
- ✨ **3D 인터랙티브 배너**: Spline 3D를 활용한 몰입감 있는 메인 배너
- 📊 **전문 분야 소개**: 20년+ 경력을 바탕으로 한 6가지 핵심 전문 분야
- 💼 **프로젝트 레퍼런스**: 실제 프로젝트 사례 및 성과 관리
- ⭐ **고객 리뷰 시스템**: 실시간 리뷰 작성 및 평점 관리
- 📧 **문의 관리 시스템**: 체계적인 고객 문의 접수 및 관리
- 📱 **완전 반응형 디자인**: 모든 디바이스에서 최적화된 경험

## 🚀 현재 완성된 기능

### 1. **홈 페이지 (Hero Section)**
- 3D 인터랙티브 배너 (Spline 통합)
- 20년+ 경력 강조
- 주요 통계 표시 (경력, 글로벌 기업, 학위)
- CTA 버튼 (상담 신청, 프로필 보기)

### 2. **프로필 섹션**
- 강사 사진 및 기본 정보
- 경력 요약
- 학력 정보 (한국해양대학교 물류대학원 석사)
- 연락처 정보

### 3. **전문 분야 섹션**
6가지 핵심 전문 분야 카드 형태로 표시:
- 해상 수출입 업무
- 컨테이너 관리
- 세관 신고 & 문서
- 운임 & 비용 관리
- 글로벌 물류 경험
- IT 스킬

### 4. **레퍼런스 섹션**
- 프로젝트 사례 목록 (DB 연동)
- 프로젝트명, 회사, 기간, 설명, 성과 표시
- 관리자 패널: 새 레퍼런스 추가 기능
- 초기 데이터 3건 포함

### 5. **리뷰 섹션**
- 고객 리뷰 카드 형태 표시 (DB 연동)
- 별점 시스템 (1-5점)
- 리뷰 작성 폼
- 자동 승인 시스템
- 초기 데이터 3건 포함

### 6. **문의 섹션**
- 문의 폼 (이름, 회사, 이메일, 전화번호, 문의 유형, 제목, 내용)
- 문의 유형: 컨설팅, 교육, 협업, 기타
- 문의 목록 표시 (관리자용)
- 상태 관리: 대기중, 처리중, 완료
- 연락처 정보 카드

### 7. **네비게이션 & UI**
- 고정 네비게이션 바
- 스크롤 시 활성 링크 자동 업데이트
- 모바일 반응형 메뉴
- 스크롤 투 탑 버튼
- 부드러운 스크롤 애니메이션

### 8. **푸터**
- 브랜드 정보
- 퀵 링크
- 연락처 정보
- 저작권 표시

## 🗄️ 데이터베이스 스키마

### 1. **inquiries (문의)**
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| name | text | 문의자 이름 |
| company | text | 회사명 |
| email | text | 이메일 |
| phone | text | 전화번호 |
| inquiry_type | text | 문의 유형 (consulting, education, collaboration, other) |
| subject | text | 제목 |
| message | rich_text | 문의 내용 |
| status | text | 상태 (pending, in_progress, completed) |
| submitted_at | datetime | 제출 시간 |

### 2. **reviews (리뷰)**
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| name | text | 리뷰어 이름 |
| company | text | 회사/직책 |
| rating | number | 평점 (1-5) |
| content | rich_text | 리뷰 내용 |
| is_approved | bool | 승인 여부 |
| created_at | datetime | 작성 시간 |

### 3. **references (레퍼런스)**
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| project_name | text | 프로젝트명 |
| company | text | 회사명 |
| period | text | 프로젝트 기간 |
| description | rich_text | 프로젝트 설명 |
| achievements | rich_text | 주요 성과 |
| is_featured | bool | 추천 여부 |
| created_at | datetime | 생성 시간 |

## 🔗 API 엔드포인트

### References
- `GET /tables/references?limit=100&sort=-created_at` - 레퍼런스 목록 조회
- `POST /tables/references` - 새 레퍼런스 추가

### Reviews
- `GET /tables/reviews?limit=100&sort=-created_at` - 리뷰 목록 조회
- `POST /tables/reviews` - 새 리뷰 추가

### Inquiries
- `GET /tables/inquiries?limit=100&sort=-submitted_at` - 문의 목록 조회
- `POST /tables/inquiries` - 새 문의 추가
